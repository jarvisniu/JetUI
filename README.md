# jet-ui

JetBrains style Vue UI components


![](screenshot.png)

## Demo

[Demo](http://niujunwei.com/jet-ui/)

## Usage

Install:
```sh
npm install jet-ui
```

Import:
```js
import Vue from 'vue'
import JetUI from 'jet-ui'

Vue.use(JetUI)
```

Use:
```html
<jt-button>OK<jt-button>
```
